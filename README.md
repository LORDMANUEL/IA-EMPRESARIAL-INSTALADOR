**RGIA MASTER** es un instalador único que despliega una plataforma completa de **Retrieval-Augmented Generation (RAG)** en un servidor Ubuntu/Debian en cuestión de minutos. Con un solo comando, transforma una máquina limpia en un ecosistema de IA potente, seguro y listo para producción, operando 100% en tu propia infraestructura.

## ¿Qué es RGIA MASTER?

Es la forma más rápida de construir un "cerebro" para tu organización. La plataforma ingiere tus documentos (manuales, bases de conocimiento, reportes) y te permite hacerles preguntas en lenguaje natural, obteniendo respuestas precisas y contextualizadas. Todo con total privacidad.

## Instalación

Para desplegar la plataforma completa, ejecuta el siguiente comando en un servidor Ubuntu/Debian limpio:

```bash
sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/tu_usuario/tu_repositorio/main/install_rag_cpu.sh)"
```
*(Nota: Reemplaza la URL con la ubicación real del script una vez que esté en tu repositorio)*

El script es 100% automatizado. Al finalizar, te proporcionará las URLs de acceso a los diferentes paneles de la plataforma.

## Documentación Completa

Toda la documentación detallada sobre la arquitectura, el uso, la configuración y las características "Pro" se genera automáticamente por el script y se encuentra en el archivo `README.md` dentro del directorio de instalación (`/opt/rag_lab/README.md`) en tu servidor.

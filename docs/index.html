Installation

Follow the steps below to get your private instance of RGIA Master up and running. Ensure you have Docker installed on your system.


# 1. Clone the repository
git clone https://github.com/rgia-master/platform.git
# 2. Navigate to the directory
cd platform
# 3. Run the installation script
./install.sh

Frequently Asked Questions

What is RGIA Master?

add

RGIA Master is a comprehensive, self-hosted AI platform that allows you to deploy, manage, and utilize a wide range of AI models within your own private infrastructure. It provides an all-in-one solution for development, monitoring, and management of AI services.

What are the system requirements for installation?

add

You will need a Linux-based server (physical or virtual) with Docker and Docker Compose installed. We recommend at least 4 CPU cores, 16GB of RAM, and 100GB of storage for the Base version. For Pro versions or for running larger models, a GPU is highly recommended.

Can I run RGIA Master on my cloud provider (AWS, GCP, Azure)?

add

Yes, absolutely. RGIA Master can be installed on any cloud instance that meets the system requirements. The installation script is designed to be cloud-agnostic.

How do I access the monitoring tools after installation?

add

Once the installation is complete, the script will output the URLs for the Grafana dashboard, Prometheus UI, and the Admin Panel. You can access them through your browser. Default login credentials will also be provided in the installation summary.

Is my data secure and private?

add

Yes. Since RGIA Master is installed entirely on your own infrastructure, all your data, models, and configurations remain under your control. We do not have access to your instance or your data, ensuring maximum privacy and security.

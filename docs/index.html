Instalación

Clona el repositorio y ejecuta el instalador. El asistente te guiará.

# 1. Clonar
git clone https://github.com/LORDMANUEL/IA-EMPRESARIAL-INSTALADOR.git
# 2. Navegar
cd IA-EMPRESARIAL-INSTALADOR
# 3. Ejecutar
sudo bash ./install_ia_empresarial.sh
